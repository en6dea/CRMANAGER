
CRMANAGER / Salary Room v21.6 — Paid Archive Logic

Что изменено:
- если задел переводится в этап «Оплачено», он автоматически переносится в архив;
- оплаченный архивный клиент продолжает учитываться в факте оплат по дате оплаты;
- при отказе или отправке в архив можно указать причину / комментарий;
- из архива можно вернуть клиента обратно в заделы кнопкой «В заделы»;
- при обычном переносе в архив ставится задача вернуться через месяц;
- оплаченные клиенты в архиве помечаются как «Учтён в факте оплат».

Как загрузить:
1. заменить index.html в корне репозитория CRMANAGER
2. сделать Commit changes
3. открыть https://en6dea.github.io/CRMANAGER/?v=21_6
