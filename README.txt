Salary Room v3 — модуль зарплаты менеджера

Что это:
- красивый desktop/mobile дашборд
- расчёт зарплаты по логике: 5% / 6% / 6% + 30 000 ₽
- учёт оплат, депозитов и ожидаемых счетов
- данные сохраняются в браузере

Как запустить локально:
1. Открой файл index.html в браузере.

Как загрузить на GitHub Pages:
1. Создай/открой репозиторий.
2. Загрузи файл index.html (и README при желании) в корень репозитория.
3. В Settings -> Pages выбери Deploy from a branch.
4. Branch: main, Folder: /root.
5. Сохрани и открой выданную ссылку.
